# Project Overview

## Introduction

This project was completed as my undergraduate major project in VLSI Design.

The project focuses on the design and analysis of a 16×16 Vedic Multiplier using Verilog HDL. The main objective was to understand the Urdhva Tiryagbhyam multiplication method and study the implementation of two different adder designs: Ripple Carry Adder (RCA) and Carry Lookahead Adder (CLA).

The design follows a hierarchical approach where smaller multiplier blocks are combined to build the complete 16×16 multiplier.

Related architecture images are available in the `images` folder.

---

## What is a Vedic Multiplier?

A Vedic Multiplier is a digital multiplier based on the **Urdhva Tiryagbhyam (Vertical and Crosswise)** method from Vedic Mathematics.

Instead of designing the complete multiplier as a single large circuit, smaller multiplier blocks are combined to create larger modules. This makes the design easier to understand, verify, and expand.

## Project Objective

The objectives of this project were:

- Study the Urdhva Tiryagbhyam multiplication method.
- Design a 16×16 Vedic Multiplier using Verilog HDL.
- Study RCA and CLA based implementations.
- Understand the simulation and analysis process of digital designs.
- Learn the basic VLSI design workflow.

## Summary

This project provided practical exposure to digital multiplier design and VLSI concepts. It helped in understanding hierarchical design, simulation analysis, and the importance of modular design in digital hardware development.
